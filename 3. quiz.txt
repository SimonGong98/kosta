1. 자바 프로그램 개발 환경에 필요한 요소 중 자바 가상 머신(JVM),
   자바 개발도구(JDK), 자바 개발 환경(JRE)의 포함 관계를 작성하시오.
   → JDK > JRE > JVM

2. 이클립스 단축키
    - 자동 임포트: Ctrl + Shift + O
    - 1주석 설정 및 해제: Ctrl + /
    - 자동 정렬: Ctrl + Shift + F
    - 폰트 축소, 확대: Ctrl + -, Ctrl + +
3. 자바의 가장 큰 특징인 플랫폼 독립성의 의미를 바르게 기술하시오.
    - JVM을 통해 서로 다른 OS에서 정상적으로 가동되도록 함.
4. 소스 파일이 "src/abc/bcd/cde/Test.java"에 위치할 때 패키지 명을 쓰시오.
    → abc.bcd.cde
5. 소스 코드의 파일명과 컴파일 이후에 생성되는 바이트 코드 파일명을 모두 쓰시오.
    → C.java, A.class, A$B.class, C.class, C$D.class, C$D$E.class
